# 💸 Monitor de Câmbio IA

Aplicação inteligente para monitoramento de cotações de moedas em tempo real, com robô de alertas via WhatsApp/Telegram e análise cambial preditiva utilizando **OpenAI GPT-4.1-mini**.

## 🚀 Funcionalidades
- **Dashboard em Tempo Real**: Cotações de USD e EUR atualizadas via ExchangeRate-API.
- **Análise com IA**: Projeções técnicas e fundamentais baseadas em notícias econômicas.
- **Robô de Câmbio**: Monitoramento em background com disparos automáticos de alertas.
- **Autenticação Segura**: Gerenciamento de usuários integrado ao **Supabase**.
- **PWA Ready**: Instalável como um aplicativo nativo no celular.

## 🛠️ Tecnologias
- [Streamlit](https://streamlit.io/)
- [OpenAI API](https://openai.com/)
- [Supabase Auth](https://supabase.com/)
- [Plotly](https://plotly.com/)

## ⚙️ Configuração Local
1. Clone o repositório.
2. Crie um ambiente virtual: `python -m venv venv`.
3. Instale as dependências: `pip install -r requirements.txt`.
4. Configure seu arquivo `.env` com as chaves necessárias.
5. Execute: `streamlit run app/main.py`.
